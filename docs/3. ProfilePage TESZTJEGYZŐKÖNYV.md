# Receptmegosztó közösségi platform – Profil oldal manuális tesztjegyzőkönyv

Tesztet végezte: Benkő Bence  
Dátum: 2025.12.11.  
Talált hibák száma: 0

## 1. Teszt célja és terjedelme
A profil oldal funkcióinak ellenőrzése bejelentkezett felhasználó esetén:
- Hozzáférés csak bejelentkezés után.
- Felhasználói alapadatok megjelenítése: név, e-mail cím.
- Saját receptek számláló helyes értéke és a receptek listázása.
- Receptkártyákon a „három soros” (overflow) menü megjelenése és működése: Edit, Delete, Details.
- Edit/Delete műveletek végrehajtása és visszajelzés, Details navigáció a recept oldalra.

## 2. Tesztkörnyezet és erőforrások
| Elem               | Konfiguráció / Leírás                           |
|--------------------|--------------------------------------------------|
| Operációs rendszer | Windows 10                                       |
| Böngésző           | Google Chrome                                    |
| Tesztelési mód     | Manuális (bejelentkezett felhasználó)           |
| Tesztelési célpont | Receptmegosztó platform (https://localhost:7224/) |
| Kiinduló oldal     | Bejelentkezés után Profil oldal

## 3. Tesztelési eljárás
1. Sikeres bejelentkezés a rendszerbe.
2. Navigáció a Profil oldalra.
3. Felhasználói adatok és receptek megjelenésének ellenőrzése.
4. Receptkártyák overflow menüjének (három soros gomb) tesztelése: Edit, Delete, Details.
5. Edit és Delete funkciók végrehajtása, majd eredmény megfigyelése (lista frissülése).
6. Details kattintás ellenőrzése: navigáció a recept részletező oldalra.
